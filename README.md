# SimpleXpress Calculator

SimpleXpress Calculator is a user-friendly web app that instantly calculates Simple Interest based on principal, rate, and time. Designed for students, it features an easy interface and responsive design for quick financial calculations.

## Features

- Enter Principal, Rate of Interest, and Time in months
- Calculates and displays Simple Interest instantly
- Mobile responsive layout

## How to Use

1. Download or clone this repository:
    ```
    git clone https://github.com/Gurunath0007/SimpleXpress-Calculator.git
    ```
2. Open `index.html` in any web browser.

## Files

- `index.html` – main webpage structure
- `style.css` – styles and layout
- `script.js` – calculation logic

## Formula

\[
Simple\ Interest = \frac{Principal \times Rate \times Time}{100}
\]
